``python-myeventhub`` - это API клиент для MyEventHub.ru


Требования
----------

Python 2.7, 3.4, 3.5. На других версиях Python не тестировалось, но, возможно, работать будет.

Установка
---------

Установить пакет через pip:

.. code:: sh

 pip install python-myeventhub
